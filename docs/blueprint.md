# **App Name**: Career Brew Tycoon

## Core Features:

- Bean Roasting: Simulate the process of roasting coffee beans with increasing efficiency.
- Store Upgrades: Purchase upgrades to increase bean roasting efficiency and unlock new features.
- Prestige System: Allow players to reset their progress for a permanent bonus, adding long-term engagement.
- Ranking System: Users can save their score to Firestore and compete on global and virtual rankings.
- AI powered naming suggestions: Generative AI suggests creative names based on store level, as a tool to provide engaging naming options

## Style Guidelines:

- Primary color: Burnt orange (#D47A4E) to evoke the warmth and aroma of coffee.
- Background color: Pale beige (#F5F5DC) to provide a soft, inviting backdrop.
- Accent color: Dark brown (#8B4513) for UI elements like buttons and progress bars.
- Headline font: 'Belleza', a sans-serif with an elegant style aligned to art and design.
- Body font: 'Alegreya', a serif offering readability for game stats and descriptions.
- Use coffee-themed icons for upgrades, achievements, and resources, keeping a consistent visual theme.
- The main layout is divided into three sections: UI display, canvas area, and upgrade panel. These sections are displayed in a top-to-bottom layout
